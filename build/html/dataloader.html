dataloader.enmo module¶

dataloader.enmo.calculate_enmo(data: DataFrame) → DataFrame[source]¶

Calculate the Euclidean Norm Minus One (ENMO) metric from raw accelerometer data.

The ENMO metric is computed as the Euclidean norm of the accelerometer vector (X, Y, Z) minus 1, with negative values set to zero. This metric is commonly used in physical activity research to quantify movement intensity.

Parameters:

data (pd.DataFrame) -- DataFrame containing accelerometer data with columns 'X', 'Y', and 'Z' representing the raw accelerometer readings along the three axes.

Returns:

The input DataFrame with an additional 'ENMO' column containing the ENMO values calculated for each row.

Return type:

pd.DataFrame

dataloader.enmo.calculate_minute_level_enmo(data: DataFrame) → DataFrame[source]¶

Resample ENMO data to minute-level by averaging over 1-minute intervals.

This function resamples high-frequency ENMO data to a minute-level resolution by taking the mean of the ENMO values within each 1-minute period.

Parameters:

data (pd.DataFrame) -- DataFrame containing timestamped ENMO data with 'TIMESTAMP' and 'ENMO' columns, where 'TIMESTAMP' is in a datetime format.

Returns:

DataFrame with minute-level averaged ENMO values, containing 'TIMESTAMP' and 'ENMO' columns where each row represents a 1-minute interval.

Return type:

pd.DataFrame

load_data()[source]¶

Loads and processes accelerometer data from CSV files in the specified directory. This method performs several transformations, including timestamp conversion, ENMO calculation, and filtering of incomplete days. It then aggregates ENMO values at the minute level and stores the result in self.enmo_per_minute.

Processing steps include:

  1. Concatenating CSV files from the input directory.

  2. Converting timestamps to POSIX format.

  3. Calculating the ENMO metric.

  4. Filtering out incomplete days.

  5. Aggregating ENMO values at the minute level.

dataloader.utility.concatenate_csv(directory_path: str) → DataFrame[source]¶

Concatenate all CSV files in a directory into a single DataFrame.

This function reads all CSV files in the specified directory that match the '*.sensor.csv' pattern, concatenates them, and returns a single DataFrame containing only the 'HEADER_TIMESTAMP', 'X', 'Y', and 'Z' columns.

Parameters:

directory_path (str) -- Path to the directory containing the CSV files.

Returns:

Concatenated DataFrame containing the accelerometer data from all CSV files, with columns 'HEADER_TIMESTAMP', 'X', 'Y', 'Z', sorted by 'HEADER_TIMESTAMP'.

pd.DataFrame

dataloader.utility.filter_incomplete_days(data_all: DataFrame) → DataFrame[source]¶

Filter out data from incomplete days to ensure 24-hour data periods.

This function removes data from the first and last days in the DataFrame to ensure that only complete 24-hour data is retained.

Parameters:

data_all (pd.DataFrame) -- DataFrame with a 'TIMESTAMP' column in datetime format, which is used to determine the day.

Returns:

Filtered DataFrame excluding the first and last days. If there are fewer than two unique dates in the data, an empty DataFrame is returned.

dataloader.utility.get_posix_timestamps(timestamps: Series, sample_rate=80) → Series[source]¶

Generate a POSIX timestamp series based on an initial timestamp and sample rate.

This function creates a series of POSIX timestamps by adding a time delta at the specified sampling rate to the initial timestamp in the series. This is useful for creating timestamped data at a consistent sampling rate.

Parameters:

  • timestamps (pd.Series) -- Series containing a single initial timestamp.

  • sample_rate (int, optional) -- Sampling rate in Hz (samples per second). Defaults to 80 Hz.

Returns:

Series of POSIX timestamps at the given sample rate.

Return type:

pd.Series
